git clone https://github.com/yourusername/tensor-nu-algebra

### Run Validation
```bash
python scripts/empirical_validation.py
```

### View Results
```bash
ls results/
# validation_report.md
# validation_results.pdf
```

## For Developers

### Setup Development Environment
```bash
pip install -e ".[dev]"
pytest
```

### Add New Probe
Edit `data/h0_measurements.csv`, add row, rerun validation.

### Extend Framework
1. Add method to `tensor_nu_algebra/operators.py`
2. Add test to `tests/test_operators.py`
3. Update documentation in `docs/api.md`
4. Run `pytest` to verify

## For Reviewers

### Reproduce Results
```bash
python scripts/empirical_validation.py
diff results/validation_report.json expected_results.json
```

### Check Reproducibility
```bash
pytest tests/
python scripts/run_full_analysis.py
```

### Verify Citations
All data sources in `data/h0_measurements.csv` include DOIs.
```
