## SECTION 1: FOUNDATIONAL MATHEMATICS

### 1.1 N/U Algebra Definition

**Carrier Set**: A = ℝ × ℝ≥0

**Elements**: (n, u) where n = nominal value, u = uncertainty (u ≥ 0)

**Operations**:

```
Addition (⊕):
  (n₁, u₁) ⊕ (n₂, u₂) = (n₁ + n₂, u₁ + u₂)

Multiplication (⊗):
  (n₁, u₁) ⊗ (n₂, u₂) = (n₁n₂, |n₁|u₂ + |n₂|u₁)

Scalar Multiplication (⊙):
  a ⊙ (n, u) = (an, |a|u)
```

**Properties** (all proven):
- Closure: ∀ operations map A → A
- Associativity: (x⊕y)⊕z = x⊕(y⊕z); (x⊗y)⊗z = x⊗(y⊗z)
- Commutativity: x⊕y = y⊕x; x⊗y = y⊗x
- Identity: (0,0) additive, (1,0) multiplicative
- Monotonicity: u₁ ≤ u₂ ⇒ all operations preserve order

**Validation Results** (70,054 tests):
- Addition vs Gaussian RSS: ratio 1.00-3.54 (conservative)
- Multiplication vs Gaussian: ratio 1.00-1.41 (conservative)
- Interval consistency: error ≤ 1.4×10⁻⁴ (machine precision)
- Monte Carlo: N/U always ≥ MC std dev (24/24 tests)

### 1.2 Observer Domain Tensor Framework

**Definition**: Each H₀ measurement exists in a 4D observer space:

```
T_obs = [a, P_m, 0_t, 0_a]

Where:
  a   = awareness (confidence level, 0-1)
```

**Epistemic Distance**:


**Tensor-Extended Uncertainty**:


**Purpose**: Encode H₀ measurements as cosmology-portable coordinates that are frame-agnostic and reproducible.

**UHA Tuple**: A = (a, ξ, û, CosmoID; anchors)

Where:
- a = scale factor (dimensionless, related to redshift via a = 1/(1+z))
- ξ = horizon-normalized position index (from Morton space-filling curve)
- û = unit direction vector (3D)
- CosmoID = cosmology fingerprint (H₀, Ωₘ, Ωᵣ, Ω_Λ)

```
R_H(a) = c ∫[0→a] da' / (a'² H(a'))

```

**Position Recovery**:

```python
# Pseudocode for complete audit with UHA integration

import math

# Physical constants
c = 299792.458  # km/s
f21 = 1.42040575177e9  # Hz

# Hubble function
def H(a, cosmo):
    H0 = cosmo["H0"]
    Or = cosmo["Omega_r"]
    Om = cosmo["Omega_m"]
    OL = cosmo["Omega_Lambda"]
    return H0 * math.sqrt(Or * a**(-4) + Om * a**(-3) + OL)

# Comoving horizon via numerical integration
def compute_RH(a, cosmo, steps=1000):
    integral = 0
    da = a / steps
    for i in range(1, steps+1):
        a_prime = i * da
        H_a = H(a_prime, cosmo)
        integral += da / (a_prime**2 * H_a)
    return c * integral  # Mpc

# UHA encoding
def create_UHA(a, r, u_hat, cosmo):
    RH = compute_RH(a, cosmo)
    xi = r / RH
    return {
        "a": a,
        "xi": xi,
        "u_hat": u_hat,
        "r": r,
        "R_H": RH,
        "H_a": H(a, cosmo),
        "CosmoID": f"{cosmo['name']}(H0={cosmo['H0']})",
        "anchors": {"c": c, "f21": f21}
    }
```
