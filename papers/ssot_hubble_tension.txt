## SECTION 1: FOUNDATIONAL MATHEMATICS

### 1.1 N/U Algebra Definition

**Carrier Set**: A = ℝ × ℝ≥0

**Elements**: (n, u) where n = nominal value, u = uncertainty (u ≥ 0)

**Operations**:

```
Addition (⊕):
  (n₁, u₁) ⊕ (n₂, u₂) = (n₁ + n₂, u₁ + u₂)

Multiplication (⊗):
  (n₁, u₁) ⊗ (n₂, u₂) = (n₁n₂, |n₁|u₂ + |n₂|u₁)

Scalar Multiplication (⊙):
  a ⊙ (n, u) = (an, |a|u)
```

**Properties** (all proven):
- Closure: ∀ operations map A → A
- Associativity: (x⊕y)⊕z = x⊕(y⊕z); (x⊗y)⊗z = x⊗(y⊗z)
- Commutativity: x⊕y = y⊕x; x⊗y = y⊗x
- Identity: (0,0) additive, (1,0) multiplicative
- Monotonicity: u₁ ≤ u₂ ⇒ all operations preserve order

**Validation Results** (70,054 tests):
- Addition vs Gaussian RSS: ratio 1.00-3.54 (conservative)
- Multiplication vs Gaussian: ratio 1.00-1.41 (conservative)
- Interval consistency: error ≤ 1.4×10⁻⁴ (machine precision)
- Monte Carlo: N/U always ≥ MC std dev (24/24 tests)

### 1.2 Observer Domain Tensor Framework

**Definition**: Each H₀ measurement exists in a 4D observer space:

```

Where:
  a   = awareness (confidence level, 0-1)
  P_m = physics model (methodology encoding, 0-1)
  0_t = temporal offset (redshift regime, normalized)

**Epistemic Distance**:

```

```

**Tensor-Extended Uncertainty**:

```

```python
# Pseudocode for complete audit

# Step 1: Load data
probes = [
    {"name": "Planck18", "H0": 67.4, "u": 0.5, "T": [0.95, 0.999, 0.0, -0.507]},
    {"name": "DES-IDL", "H0": 67.19, "u": 0.65, "T": [0.9, 0.333, 0.016, -0.31]},
    {"name": "SH0ES", "H0": 73.04, "u": 1.04, "T": [0.8, 0.01, -0.048, 0.514]},
    {"name": "TRGB", "H0": 69.8, "u": 2.5, "T": [0.75, 0.01, -0.048, 0.534]},
    {"name": "TDCOSMO", "H0": 73.3, "u": 5.8, "T": [0.7, 0.231, -0.048, -0.073]},
    {"name": "Megamaser", "H0": 73.5, "u": 3.0, "T": [0.85, 0.01, -0.048, 0.439]}
]

# Step 2: Aggregate groups
early = probes[0:2]
late = probes[2:6]

# Step 3: Inverse-variance weighting
def weighted_merge(group):
    weights = [1/p["u"]**2 for p in group]
    W = sum(weights)
    w_norm = [w/W for w in weights]
    
    n = sum(w * p["H0"] for w, p in zip(w_norm, group))
    u_sq = sum(w**2 * p["u"]**2 for w, p in zip(w_norm, group))
    u = sqrt(u_sq)
    
    T = [sum(w * p["T"][i] for w, p in zip(w_norm, group)) for i in range(4)]
    
    return {"n": n, "u": u, "T": T}

early_agg = weighted_merge(early)
late_agg = weighted_merge(late)

# Step 4: Epistemic distance
delta_T = sqrt(sum((early_agg["T"][i] - late_agg["T"][i])**2 for i in range(4)))

# Step 5: Tensor-extended merge
disagreement = abs(early_agg["n"] - late_agg["n"])
u_expand = (disagreement / 2) * delta_T
u_merged = (early_agg["u"] + late_agg["u"]) / 2 + u_expand
n_merged = (early_agg["n"] + late_agg["n"]) / 2

# Step 6: Concordance check
early_interval = [early_agg["n"] - early_agg["u"], early_agg["n"] + early_agg["u"]]
late_interval = [late_agg["n"] - late_agg["u"], late_agg["n"] + late_agg["u"]]
merged_interval = [n_merged - u_merged, n_merged + u_merged]

early_contained = (merged_interval[0] <= early_interval[0] and 
                   early_interval[1] <= merged_interval[1])
late_contained = (merged_interval[0] <= late_interval[0] and 
                  late_interval[1] <= merged_interval[1])

gap = max(0, late_interval[1] - merged_interval[1])
systematic_needed = gap / 2

# Output results
print(f"Merged H0: {n_merged:.2f} ± {u_merged:.2f} km/s/Mpc")
print(f"Epistemic distance: {delta_T:.4f}")
print(f"Early contained: {early_contained}")
print(f"Late contained: {late_contained}")
print(f"Systematic needed: {systematic_needed:.3f} km/s/Mpc")
```
